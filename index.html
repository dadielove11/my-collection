GitHub Gist 同步

用 GitHub Gist 在多台电脑间同步数据。前往 GitHub → Settings → Developer settings → Personal access tokens → Generate new token，勾选 gist 权限。首次推送会自动创建私有 Gist，Gist ID 自动保存。

推送后可在 gist.github.com 查看，URL 末尾的 32 位字符串即为 ID
